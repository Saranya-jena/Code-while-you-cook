Compilation and Execution

We know, to make perfect dish we need to add each and every ingredient carefully, Imagine how will it taste if we add salt instead of sugar in the tea!!

So in the same way we we need to write a program even more carefully not a single character should be missed!


Can you eat your lunch without cooking the raw vegetables,rice and pulses?


You need to cook the ingredients first i.e convert from raw form to cooked form. Similarly these computers are are not smart enough to understand the code you write so the code you write needs to be converted/translated from your programming language(C++) to computer language i.e. binary language.And this process of coverting human level language to machine level language is known as compilation and it is done by compiler.


Another one, before you let your family taste the delicious lunch that you are going to prepare you need to cook first, in the same way to use your program that you just coded you need to run/execute it first.

Congratulations
Now you know what is Compilation and execution !
